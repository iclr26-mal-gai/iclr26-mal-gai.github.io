# ICLR 2026 Workshop on Multi-Agent Learning and Generative AI

This repository hosts the website for the ICLR 2026 Workshop on Multi-Agent Learning and Its Opportunities in the Era of Generative AI.

## Website Structure

- `index.html`: The main page containing all workshop information.
- `style.css`: The stylesheet for the website.

## How to Update

1. **Edit Content**: Open `index.html` and modify the text sections (About, Call for Papers, Speakers, Schedule, Organizers).
2. **Edit Styles**: Open `style.css` to change colors, fonts, or layout.
3. **Images**: To add speaker images, place them in an `img/` folder and update the `src` attributes in `index.html`, replacing the font-awesome icons.
